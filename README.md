# URP txAdmin Theme

Custom txAdmin theme voor Utrecht Roleplay.

## Installatie
1. Download de laatste release
2. Unzip naar de server
3. Run `bash install.sh`

Of voeg toe aan Pterodactyl startup command:
```
cd /home/container && curl -sL https://github.com/Nica0355/urp-txadmin-theme/releases/latest/download/urp-theme.tar.gz | tar xz && bash install.sh
```
